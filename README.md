📌 Project Overview

The Online Exam Proctoring System is an AI-powered application designed to conduct secure online examinations. It uses Python and OpenCV to monitor candidates through their webcams in real time, helping prevent malpractice and maintain exam integrity.

The system automatically tracks exam duration, detects suspicious activities, generates alerts, and submits exams when the allotted time expires. This provides a fair and reliable examination environment for educational institutions and organizations.

⚙️ Installation
1. Clone the Repository
git clone https://github.com/your-username/Online-Exam-Proctoring-System.git
2. Navigate to the Project Directory
cd Online-Exam-Proctoring-System
3. Install Dependencies
pip install -r requirements.txt
4. Run the Application
python main.py
📖 How It Works
Candidate starts the exam.
Webcam access is enabled.
The system continuously monitors the candidate.
Face detection verifies the candidate's presence.
Any suspicious activity is flagged.
Exam timer tracks remaining time.
Exam is automatically submitted when time expires.
Results are stored and displayed.
